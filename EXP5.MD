**Perform Following Queries using Employee Table (Aggregate Functions & String Functions)**

---

**Query 1:** Display the total number of employees working in the company.

```sql
SELECT COUNT(*) AS TOTAL_EMPLOYEES
FROM EMPLOYEE;
```

**Output:**
```
+-----------------+
| TOTAL_EMPLOYEES |
+-----------------+
|              14 |
+-----------------+
1 row in set (0.001 sec)
```
